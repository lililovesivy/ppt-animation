# PPT 智能动画生成器

为法律类口播视频PPT自动添加元素级动画，基于PowerPoint原生XML实现，确保100%兼容性。

## ✨ 特性

- **元素级动画**：每个元素独立入场，而非整页淡入淡出
- **智能分类**：自动识别9种元素类型（标题、正文、重点、图片、图标、形状、线条、高亮框、背景）
- **类型匹配**：不同类型元素使用不同动画风格，层次分明
- **智能方向**：根据元素位置自动选择最佳飞入方向（支持8方向）
- **节奏明快**：符合口播视频节奏，不拖沓，不花哨
- **原生兼容**：直接修改PPT内部XML，所有版本PowerPoint均可正常播放

## 🚀 快速开始

### 环境要求

- Python 3.6+
- python-pptx 库

```bash
pip install python-pptx
```

### 使用方法

1. 修改脚本头部的配置：
```python
original_pptx = '你的原始PPT文件路径.pptx'
output_pptx = '输出文件路径.pptx'
```

2. 运行脚本：
```bash
python scripts/gen_anim.py
```

3. 打开生成的PPT，点击播放即可看到元素级动画效果

## 🎨 元素动画映射规则

| 元素类型 | 动画类型 | 方向规则 | 触发方式 |
|---------|---------|---------|---------|
| 背景图 | 无动画（固定显示） | - | - |
| 标题文字 | 擦除 | 自左向右 | 点击触发 |
| 正文文字 | 擦除 | 自左向右 | 点击触发 |
| 重点文字 | 飞入 | 从下方弹入 | 点击触发 |
| 内容图片 | 飞入 | 从最近的边缘/角落飞入 | 点击触发 |
| 小图标 | 飞入 | 从最近的边缘飞入 | 点击触发 |
| 装饰形状 | 飞入 | 从最近的边缘飞入 | 点击触发 |
| 高亮框 | 擦除 | 自左向右 | 点击触发 |
| 线条 | 擦除 | 从左到右 / 从上到下 | 点击触发 |

## ⚙️ 技术原理

基于PowerPoint原生动画XML结构，直接修改PPT内部XML文件实现。无需借助第三方工具，生成的PPT保持完全可编辑。

### 支持的动画类型

1. **飞入动画 (Fly In)** - presetID: 2
   - 8个方向（含对角线）
   - 适用于图片、形状、图标
   - 有动感，适合强调元素

2. **擦除动画 (Wipe)** - presetID: 22
   - 4个方向（上/下/左/右）
   - 适用于文字、高亮框、线条
   - 平滑自然，适合文字内容

## 📝 注意事项

1. **兼容性优先**：只使用PowerPoint标准动画，确保跨版本兼容
2. **内容不变**：只添加动画，不修改任何文字、图片、布局
3. **适度原则**：动画服务于内容表达，不过度炫技
4. **节奏适配**：法律类口播节奏偏快，动画时长控制在0.5-1秒

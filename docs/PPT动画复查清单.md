# PPT 动画项目 — 复查清单

> 以下问题在历史上出现多次，交付前必须逐页检查。

## 交付前强制检查项（逐页）

### 1. 分割线/装饰线时机
- [ ] 标题分割线是否在标题出现之后才出现？（不能始终可见）
- [ ] 卡片内部竖线是否在所有文字/图片之后才出现？
- [ ] 卡片顶部色条/装饰条是否在卡片背景之后、内容之前出现？（不能始终可见）
- [ ] 所有非内容元素（线条、色块、装饰）都有 entrance 动画

### 2. 文字遮挡
- [ ] **字号差异大的相邻文本框**：检查 y 坐标间距
  - 14pt→28pt 字号转换：间距 ≥ 6pt
  - 28pt→12pt 字号转换：间距 ≥ 6pt
- [ ] 文本框高度是否足够容纳实际字号（建议 h ≥ font_size × 1.2）
- [ ] 首次交付前必须用 slide_get_page_info 检查所有相邻文本框的 y 坐标

### 3. 同层级一致性
- [ ] 同一层级的文字（标签/数值/描述）字号、颜色是否一致
- [ ] 不同页面间的同类元素字号、颜色是否一致
- [ ] 注意：装饰性大字（如左侧 40pt 显示）和标准数值（如右侧 28pt）是不同的层级

### 4. 动画次数
- [ ] 每个形状只应该有 entrance 动画（每个形状一个 entrance）
- [ ] 不能出现 exit 动画替代 entrance 动画
- [ ] 调用 slide_get_page_info 后，逐条检查 category 字段
- [ ] 第一个动画（index 0）的 category 必须是 "entrance"

### 5. 验证方法
每次修复后必须调用 `slide_get_page_info` 验证：
- anim_count 是否等于预期数量
- index 0 的 category 是否为 "entrance"
- 新增的形状是否有对应用时机的动画
